# Med Amine Balti Portfolio

Production portfolio and case-study hub for Med Amine Balti, a senior full-stack engineer building SaaS, fleet platforms, AI tools, dashboards, marketplaces, and high-conversion product websites.

- Live site: https://aminebdev.vercel.app
- GitHub profile: https://github.com/aminebalti55
- LinkedIn: https://www.linkedin.com/in/aminebdev/

## What This Repo Shows

This is not a starter portfolio. It is a recruiter/client-facing product surface built to prove delivery quality:

- Production-grade case studies for SaaS, AI automation, fleet, marketplace, finance, and service-business work.
- SEO metadata, Open Graph support, Vercel Analytics, and Speed Insights.
- Responsive UI with dark mode, motion, reusable sections, and project-specific detail pages.
- Real project screenshots and delivery snapshots instead of generic template content.

## Featured Work

## Tech Stack

- Framework: Next.js 16, React 19, TypeScript
- Styling: Tailwind CSS 4, custom components, dark mode
- UI/motion: Radix UI, Framer Motion, Lucide icons
- Observability: Vercel Analytics, Vercel Speed Insights
- Tooling: ESLint, PostCSS, npm

## Local Development

```bash

```
